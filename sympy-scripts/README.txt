This python script has been tested with the following settings:

Python 2.7 
Sympy 0.7.1 
Bottle 0.12.8

The default script runs at port 8080. To change the port number modify 
the following line in EqScoringWebService.py to whichever port you want 
to use:

run(host='localhost', port=8080, debug=True)
